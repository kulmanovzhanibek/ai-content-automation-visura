# Reels Machine

Automated short-form video pipeline: reference → voiceover + images → Kling
timelapse transitions → Remotion montage with captions → Telegram.

## Pipeline (per job)
1. Script + N image prompts (generated by Claude, single consistent style)
2. gen-images: Gemini `gemini-3.1-flash-image` (Nano Banana 2), aspect 9:16 → PNG
3. tts: ElevenLabs with character-level timestamps → voice.mp3 + timestamps.json
   (fixed voice, ELEVENLABS_VOICE_ID from .env — never regenerate the voice)
4. captions: convert char timestamps → word tokens for @remotion/captions
5. Kling (via MCP): file_upload each PNG → image_to_video pairs → poll → clips
6. Remotion: clips back-to-back + voice + TikTok-style captions → 1080x1920 mp4
7. telegram: sendVideo to channel

## Hard rules
- Kling image_to_video: model `kling-video-v2_5`, inputs `first_image`+`tail_image`,
  args `enable_audio=false`, `duration=5`. NEVER use kling3.0-turbo for transitions
  (no tail_image support).
- Kling file_upload accepts local file paths (agent-side, MCP only).
- Images MUST be 9:16 — Kling video inherits aspect ratio from the input image.
- N images → N-1 clips (pair i uses image[i] as first, image[i+1] as tail).
- Voice length must match total clip duration = (N-1) * 5s. Adjust script length or N.
- All work under jobs/<job_id>/. Idempotent: skip steps whose output already exists.
- Kling generations cost credits — never fire trial/throwaway jobs.
- Remotion: solo/individual = free license.

## APIs (verify exact params against current docs before coding)
- Gemini image: @google/genai, Interactions API, model gemini-3.1-flash-image,
  set aspect ratio 9:16 via image_config. Output is base64 → write PNG.
- ElevenLabs: /v1/text-to-speech/{voice_id}/with-timestamps → audio + alignment
  (characters[], character_start_times_seconds[], character_end_times_seconds[]).
- @remotion/captions: build Caption tokens {text,startMs,endMs,timestampMs},
  then createTikTokStyleCaptions for word-by-word highlight pages.
- Telegram: Bot API sendVideo (multipart), TELEGRAM_BOT_TOKEN + TELEGRAM_CHAT_ID.

## Env (.env)
GEMINI_API_KEY, ELEVENLABS_API_KEY, ELEVENLABS_VOICE_ID,
TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID

## Discipline
Build the vertical slice first (one video end-to-end) before any automation,
templating, or Telegram trigger. Do not build ahead of the current phase.
